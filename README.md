# AI Question → Google Docs Extension

A Chrome extension that allows teachers to quickly export AI-generated math questions into Google Docs with correct formatting.

## Problem

Many teachers generate math problems using AI tools.  
However, when copying the text directly into Google Docs, the formatting often breaks.

## Solution

This extension adds a simple button that allows users to instantly transfer AI-generated questions into Google Docs with proper formatting.

## Features

• One-click export to Google Docs  
• Preserves formatting  
• Designed for teachers and educators  

## Tech Stack

JavaScript  
Chrome Extension API

## Installation

1. Clone the repository
2. Open Chrome Extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the extension folder
